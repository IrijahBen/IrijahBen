# Movement of the Planet

This repository contains code for analyzing and visualizing the movement of planets and moons using a dataset from Kaggle. The main feature is an animation created from images of planets and moons.

## Repository Structure

- `notebooks/`: Contains the Jupyter notebook used for the analysis.
- `data/`: Folder for datasets used in the analysis (if any).
- `requirements.txt`: List of Python packages required to run the notebook.
- `README.md`: Project description and instructions.

## Dataset

The dataset used in this project is the "Planets and Moons Dataset - AI in Space" from Kaggle. You can find it [here](https://www.kaggle.com/datasets/planets-and-moons-dataset-ai-in-space).
